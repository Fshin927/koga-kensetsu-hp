```
vNN/
  index.html        … 1ページ構成（セクション: ヒーロー/事業内容/施工実績/簡単見積もり/会社概要(Googleマップ含む)/お問い合わせ）
```

## 必須機能（全サイト共通）
1. **レスポンシブ対応**: スマホ(〜600px)/タブレット/PC。ハンバーガーメニュー実装。
2. **施工実績**: `js/works-data.js` の配列からJSで描画。形式は厳守:
   ```js
   // ★施工実績の追加方法★
   // 下の配列に1件追加するだけ。image は images/ に入れたファイル名（例: "work1.jpg"）。
   // image を "" にすると自動でプレースホルダー画像が表示されます。
   const WORKS = [
     { title: "○○ビル足場仮設工事", category: "足場仮設", place: "東京都府中市", date: "2025年10月", description: "地上8階建てビルの外壁改修に伴う枠組足場の設置。", image: "" },
     // …計6件程度のダミーデータを入れる
   ];
   ```
   画像が "" の場合は カテゴリ名入りのSVGデータURIまたはCSSグラデーションのプレースホルダーを表示。カードクリックで詳細（モーダル等）を出すと尚良い。
3. **お問い合わせフォーム**: 名前/会社名(任意)/電話/メール/内容。JSバリデーション付き。
   `js/main.js` 冒頭に `const CONTACT_ENDPOINT = "";` を定義し、コメントで「Formspree等のURLを設定すると実送信されます」と説明。未設定時は送信ボタンで「(デモ) 送信内容を確認しました。実運用時はCONTACT_ENDPOINTを設定してください」等の成功風メッセージを表示。
4. **簡単見積もりツール**: 工事種別（足場仮設/土工事/コンクリート工事/外構工事/解体工事 等）、規模（㎡や階数等の数値入力）、オプション（夜間工事、急ぎ 等）を選ぶと概算金額レンジを即時表示。計算ロジックはJSで適当な単価でよい。「※あくまで目安です。正式なお見積もりはお問い合わせください」の免責を必ず表示。
5. **Googleマップ**: APIキー不要のiframe埋め込みを使用:
   `<iframe src="https://www.google.com/maps?q=東京都府中市小柳町1-20-1&output=embed" ...>`
   本社と八王子拠点の2つを載せること（タブ切替か並べて表示）。
6. 会社概要テーブル（上記会社情報をすべて記載）、フッターに許可番号を記載。

## 品質基準
- `<html lang="ja">`、適切な title / meta description / OGP。
- 文章はすべて自然な日本語で。lorem ipsum 禁止。キャッチコピーや挨拶文も本物らしく書く。
- 電話番号は `tel:` リンクにする。
- デザインは指示されたテーマに沿って高品質に作り込む。手抜きの白ページにしない。
